Metrics visualization toolkit for Grid5000

Example Source

This set of scripts allows you, after an experiment on the Grid5000 platform, to fetch per-machine metrics and to see them nicely stacked, thanks to Mike Bostock's D3.js . Initially created to perform Hadoop jobs forensics (hence the "Counters" side), the Hadoop part can be bypassed if you just want to see how the ressource usage among your cluster evolved during an experiment.

Release in progress - more details coming soon !
